# Add Admin UI Screenshot

For the README, add a screenshot of the Admin UI at `docs/screenshot.png`:

1. Run the emulator: `make run`
2. Open http://localhost:8085/ui/queues
3. Create a queue and a task
4. Take a screenshot
5. Save as `docs/screenshot.png`

Then the README will display it. Remove the `docs/` folder from .gitignore if you add the screenshot.
